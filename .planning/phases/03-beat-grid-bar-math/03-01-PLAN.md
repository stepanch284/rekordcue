Create `bar_math.py` in the project root with these exact functions:

**Module docstring:** "bar_math.py — Beat grid math for RekordCue. Pure functions operating on beat grid arrays from waveform.get_beat_grid(). No ANLZ file I/O here except compute_bar_energies() which reads PWV3/PWAV."

**Constants:**
```python
DNB_BPM_MIN = 155.0
DNB_BPM_MAX = 185.0
```

**Function 1: validate_bpm_range(content) -> tuple[float, bool]**
- `bpm = content.BPM / 100.0`
- `is_valid = DNB_BPM_MIN <= bpm <= DNB_BPM_MAX`
- Return `(bpm, is_valid)`

**Function 2: snap_to_8bar_boundary(position_ms: float, bar_times_ms: np.ndarray) -> tuple[int, int]**
- `diffs = np.abs(bar_times_ms - position_ms)`
- `nearest_bar_idx = int(np.argmin(diffs))`
- `eight_bar_idx = round(nearest_bar_idx / 8) * 8`
- `eight_bar_idx = min(max(eight_bar_idx, 0), len(bar_times_ms) - 1)`
- Return `(int(bar_times_ms[eight_bar_idx]), eight_bar_idx)`

**Function 3: compute_bar_energies(db, content, bar_times_s: np.ndarray) -> np.ndarray**
- Try PWV3 from .EXT first: `db.get_anlz_path(content, 'EXT')`, check Path exists, `AnlzFile.parse_file()`, check `'PWV3' in anlz_ext.tag_types` (NEVER use `in anlz` — infinite recursion bug), `anlz_ext.get_tag('PWV3').get()[0].astype(float)`
- Fallback to PWAV from .DAT: same pattern with `'PWAV'`
- If neither available: `raise ValueError(f"No waveform data available for track {content.ID}")`
- Call `_bars_from_amplitude()` helper for the actual computation

**Function 4 (private): _bars_from_amplitude(amp: np.ndarray, bar_times_s: np.ndarray, track_len_s: float) -> np.ndarray**
- `n = len(amp)`
- For each bar i: `idx_start = int(bar_times_s[i] / track_len_s * n)`, `idx_end = int(bar_times_s[i+1] / track_len_s * n)` if i+1 < len else n
- `energies[i] = float(np.mean(window))` if window non-empty, else 0.0
- Return float64 array of shape (n_bars,)

**Imports:** `numpy as np`, `from pathlib import Path`, `from pyrekordbox import AnlzFile`

After creating bar_math.py, run the test suite. All tests must pass (GREEN phase).
